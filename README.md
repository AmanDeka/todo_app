# Todo App
<br>
A multipage todo application. Made for a task given by W3DEV

## Packages Used
<br>

### Backend
- express-js
- prisma - Database ORM
- passport-js - Authentication
- bcrypt - password hashing
- mongodb

### Frontend
- react-js
- tanstack-query - Query Management (caching)
- react-router - Single app routing
- tailwind-css
