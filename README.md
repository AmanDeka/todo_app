# Todo App :pencil:
<br>
A multipage todo application :book:. <br>

## Packages Used

### Backend
- express-js
- prisma - Database ORM
- passport-js - Authentication
- bcrypt - password hashing
- mongodb

### Frontend
- react-js
- tanstack-query - Query Management (caching)
- react-router - Single app routing
- tailwind-css
